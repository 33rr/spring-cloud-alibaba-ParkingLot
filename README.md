# spring-cloud-alibaba-ParkingLot
基于spring-cloud-alibaba套件的微服务架构的停车场实战案例

关键技术点应用：
1. 服务注册. 服务发现——nacos
2. 分布式配置——nacos
3. 分布式锁——redission
4. 分布式事务——seata
5. 接口限流. 熔断. 降级——sentinel/hystrix
6. 统一网关——spring-cloud-gateway
7. 代码简化——lombok,mybatis-generator,mybatis-plus
8. 异步消息处理——rocketmq
9. 分布式缓存——redis
10. 客户端负载均衡——openFeign
